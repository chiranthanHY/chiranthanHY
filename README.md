### 🔭 Featured Projects

**[TrendForgeAI](https://github.com/chiranthanHY/TrendForgeAI)** — GenAI SaaS Platform
Automated content generation + viral trend prediction across LinkedIn, X, YouTube, and Google Trends, built with Google Gemini, FastAPI (Celery + Redis), and a Next.js dashboard with A/B testing.

**[BetterUptime](https://github.com/chiranthanHY)** — Distributed Website Monitoring System
Real-time uptime monitoring with 30s health checks and instant alerts. Turborepo monorepo (Bun + Prisma + PostgreSQL + Redis + Docker) with JWT auth and worker services.

**[DoodleArt](https://github.com/chiranthanHY)** — Real-Time Collaborative Drawing App
Live multi-user drawing and chat over WebSockets, built as a Turborepo monorepo (Next.js + React + Express + Prisma + PostgreSQL).

**[AutoCrew](https://github.com/chiranthanHY/AutoCrew-AI)** — Multi-Agent Automation Platform
LangGraph-based system where Planner, Researcher, Critic, and Executor agents collaborate autonomously, with tool calling, memory, and human-in-the-loop workflows.

**[Federated Cancer Detection](https://github.com/chiranthanHY/Privacy-Preserving-Cancer-Detection-using-Federated-Learning-with-Synthetic-Data-Augmentation)** — Privacy-Preserving ML (Group Project)
CNN-based cancer detection from CT/MRI scans trained across simulated hospital nodes via Federated Averaging. Built the data pipeline and CNN architecture.
